11일차 개발 일지
개발 목표

단일 적 대상 공격 카드가 실제 피해를 적용하고, 적 체력 UI 및 전투 불능 상태까지 정상 갱신되도록 구현한다.

구현 내용
CardEffectType을 추가하여 카드 효과 종류를 관리하도록 구성
CardData에 효과 종류와 기본 효과 수치 추가
CardInstance가 전투 중 사용할 현재 효과 수치를 보관하도록 구성
BattleCardEffectResolver를 추가하여 카드 효과 적용 로직 분리
단일 적 대상 선택 후 카드 소비, AP 차감, 피해 적용 순서 연결
공격 카드 피해량이 선택한 적의 체력에 적용되도록 구현
체력 변경 이벤트를 통해 적 체력 UI가 즉시 갱신되도록 연결
체력이 0이 된 적을 전투 불능 상태로 처리
전투 불능 적은 이후 카드 대상 선택 목록에서 제외되도록 확인
테스트 공격 카드 TEST STRIKE에 피해 효과와 피해량 설정
테스트 결과
공격 카드 선택 및 USE CARD 동작 확인
적 선택 후 AP 차감 확인
사용 카드가 Hand에서 Discard로 이동하는 것 확인
TEST STRIKE 사용 시 적 체력이 감소하는 것 확인
적 체력 Slider 및 수치 텍스트 갱신 확인
적 체력 0 도달 시 전투 불능 처리 확인
전투 불능 적 대상 재선택 불가 확인
Console 빨간색 오류 0개 확인
